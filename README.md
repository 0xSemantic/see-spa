# 🌿 Serene Spa Booking System

Full-stack-free spa booking app with Firebase, EmailJS, dummy payments & admin portal.

## Quick Start
1. download the source code, and unzip it
2. change directory into the source code the path where you see package.json
2. run "npm install" on your command interface
3. run "npm run dev" visit → http://localhost:5173 to see the app
